# 📊 Data

All raw and enriched data files produced by the intelligence pipeline.

| File | Description |
|------|-------------|
| `competitor_analysis.csv` | Master output from the Darwin Engine. 28 candidates with subscriber counts, fitness scores across 7 dimensions, generation eliminated, and webinar/offer signals. |
| `deep_enriched.csv` | Enriched version with real URLs added — website, LinkedIn, Instagram, course pages, and webinar pages discovered via automated Google search. |
| `nuno_vs_competitors.csv` | Side-by-side comparison matrix: Nuno's benchmark scores vs Top 3 finalists across all metrics. |
| `top3_competitors.txt` | First-pass Top 3 report generated by the Python script. Name, fitness score, channel URL, offer detected, webinar strategy. |
| `top3_intelligence.txt` | 🔥 The crown jewel. Full deep-stalk intelligence report on Nick Saraev, Riley Brown, and Nate Herk — generated via Claude Code + RivalSearchMCP. Includes all social URLs, pricing, revenue estimates, press mentions, podcast appearances, and business model breakdown. |
| `session_transcript.txt` | Raw Claude Code terminal session log from the RivalSearchMCP research run. |
